About Us

Artware는 경희대학교에서 모인 창의적인 팀으로, 기술과 디자인의 조화를 통해 사용자 중심의 프로젝트를 만들고 있습니다. 이번 해커톤에서는 AI와 음악을 결합한 창작 플랫폼을 개발합니다.

🚀 Project: MuseNote

  • 기획: 창작자를 위한 AI 음악 노트 플랫폼
  • 기능: 작곡 추천, 실시간 편집, 협업, 저장
  • 기술 스택: React, Flask, TensorFlow, PostgreSQL
  • 코드 저장소: GitHub
